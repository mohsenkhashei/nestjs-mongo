## Description

simple [Nest](https://github.com/nestjs/nest) framework with mongoDB

## Installation

```bash
$ npm install
```

## Running the app

```bash
# develpment with running mongo
$ npm run dev

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
